A secure Digital Health Wallet that allows users to store, track, and share their medical vitals and health reports anytime, anywhere.


Built as part of the 2care.ai hiring assignment using Node.js, Express, and SQLite.

## Frontend (ReactJS)

A minimal React frontend is included to demonstrate:
- User login
- Vitals entry
- Report upload
- API integration with JWT authentication

Frontend runs on port 3000 and connects to Node.js backend on port 8080.
